ShowCab is a user-centered UX design project that explores an integrated platform for event discovery, ticket booking, and transportation planning.

## Problem
Users attending events face fragmented workflows across multiple platforms for tickets, transport, and coordination.

## Solution
ShowCab unifies event discovery, ticket booking, and transportation planning into a single, streamlined user experience.

## Contributions
- User research and persona development
- Prototype design and interaction flow mapping
- Usability testing and evaluation
- Design refinement
